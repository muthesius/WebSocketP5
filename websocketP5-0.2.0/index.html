websocketP5

A library by jens alexander ewald for the programming environment processing. Last update, 11/22/2015.

A Simple and still rought wrapper for a WebSocketServer imbracing the webbit package.

Download

Download websocketP5 version 0.2.0 in .zip format.

Installation

Unzip and put the extracted websocketP5 folder into the libraries folder of your processing sketches. Reference and examples are included in the websocketP5 folder.

Reference. Have a look at the javadoc reference here. a copy of the reference is included in the .zip as well.

Source. The source code of websocketP5 is available at GitHub, and its repository can be browsed here.
